# Anxiety and Depression - APHRC
This repository now contains the code to run phenotype diagnostics for anxiety and depression. It includes two subfolders:

1. diagnostics_code: code to run the diagnostics for the cohorts of interest
2. diagnostics_shiny: the corresponding Shiny app to visualise the output

NOTE: At this stage, we have only written code for the diagnostics, so the folders related to the study code and study shiny are empty.
